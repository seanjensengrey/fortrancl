This table contains the supported scalar arguments and the equivalent OpenCL kernel language types. Other types are not supported.

| **Fortran type**   | **Kernel type** |
|:-------------------|:----------------|
| character (dev)    | char            |
| integer(4)         | int             |
| integer(8) (dev)   | long            |
| real(4)            | float           |
| real(8)            | double          |
| complex(4) (dev)   | float2          |
| complex(8) (dev)   | double2         |
